# App_Table_Calculator-

‎1. App Search System:
‎यह फ़ीचर यूज़र को दिए गए ऐप्स की लिस्ट में से किसी भी ऐप को जल्दी और आसानी से सर्च करने की सुविधा देता है। जैसे ही यूज़र सर्च बॉक्स में टाइप करता है, ऐप्स फिल्टर होकर दिखाई देने लगते हैं।
‎
‎2. Table Generator:
‎इसमें यूज़र किसी भी नंबर को इनपुट कर सकता है, और उसी नंबर का पूरा गुणा पहाड़ा (table) तुरंत दिखाया जाता है।
‎
‎3. Calculator:
‎यह साधारण कैलकुलेटर है जो दो अंकों को जोड़ , घटाना , गुणा ,भाग का काम करता है। यूज़र दोनों नंबर डालकर उनके परिणाम देख सकता है।
‎
